# Start Here

Thought for a while is a public, self-directed, graduate-level applied program at the intersection of:
- AI in healthcare (high-stakes decision support)
- XR + robotics (training and procedural systems)
- drug discovery + longevity
- precision genomics
- multi-agent systems and orchestration
- global medical threats
- neurotechnology / BCI

This hub exists so the work is **auditable**, **reproducible**, and **useful** to others.
